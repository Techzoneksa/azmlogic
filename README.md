# منصة عزم للتشغيل اللوجستي

تجربة المرحلة الأولى لمنصة تشغيل لوجستي عربية بالكامل لإدارة الطلبات والطرود والشركاء والمناديب والمركبات والتوزيع ووثائق النقل وجاهزية الربط المستقبلي.

## التقنية

- Next.js App Router
- React
- TypeScript
- CSS مخصص بتخطيط RTL
- حالة عرض محلية بدون اعتماد على خادم خارجي في المرحلة الأولى
- أيقونات `lucide-react`

## الأوامر

```bash
npm install
npm run dev
npm run lint
npm run build
npm run start
```

## الأدوار التجريبية

- المدير العام
- مدير العمليات
- منسق التشغيل
- مسؤول الامتثال
- المندوب
- مشاهد التقارير

## المسارات الرئيسية

تشمل التجربة مسارات تسجيل الدخول ولوحة التحكم والشركاء والطلبات والطرود والتوزيع والمناديب والمركبات ونقاط الاستلام ومحاولات التسليم والمرتجعات ووثائق النقل وجاهزية بيان والتقارير وسجل الأنشطة والإعدادات وواجهة المندوب.

## جاهزية الإنتاج

- الواجهة عربية بالكامل واتجاهها RTL.
- ملفات التطبيق والأصول داخل المستودع.
- متغيرات البيئة موثقة في `.env.example`.
- لا يتم تضمين أسرار أو مفاتيح اعتماد داخل الواجهة.
- وثائق النقل في العرض محفوظة كمسودات ولا تمثل اعتمادا رسميا.
- المشروع مهيأ للبناء والتشغيل عبر أوامر npm القياسية.

## متغيرات البيئة

راجع `.env.example` وملف النشر في `docs/HOSTINGER_DEPLOYMENT.md`.
